


## Create Tag
```bash
git checkout master
git tag -a v3.1.0 -m "Version 3.1.0"
git push origin v3.1.0
git tag
```

## Create Release
1. ```Open GitHub``` -> ```Tags``` -> ```Releases``` -> ```Draft a new release``` 
2. ```Select tag (v3.1.0)```
3. ```Release title (BibleBrainRing v3.1.0)```
4. ```Attach binaries ...```
